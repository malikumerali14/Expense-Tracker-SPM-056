# Expense Tracker System

## Project Overview
The Expense Tracker is a web-based financial management tool designed to help users track their daily income and expenses, categorize transactions, and visualize their monthly spending habits.

## System Features & Scope
* **User Authentication:** Secure registration and login functionalities.
* **Transaction Logging:** Users can Add, Read, Update, and Delete (CRUD) expense records.
* **Category Management:** Categorization of expenses (e.g., Food, Utilities, Entertainment, Rent).
* **Analytics & Reports:** A simple dashboard displaying charts of monthly spending breakdowns.
